
# Drudge Proyect

Coleccion de Skills, Agentes, etc para el uso del CDC

OBJETIVO: Creacion colaborativa de diferentes agentes para que usemos dentro del CDC, no solo desarrollo o QA, sino pentesting o etc.


## Optimizations

Nombres opcionales

«Escoria/Mugre» (Muck/Slime): Utilizado para degradar completamente la humanidad del individuo.

«Drudge» o «Skivvy»: Términos que significan una persona que hace el trabajo más sucio y servil, con un tono despectivo.

«Mascota» (Pet): Implica una relación de servidumbre disfrazada de afecto, a menudo usado en humor negro para mostrar una dinámica tóxica.

«Fustilarian»: Un insulto antiguo (muy shakesperiano) que define a un sinvergüenza apestoso y sin valor.

«Cumberworld»: Alguien que solo existe para ocupar espacio y consumir recursos.
